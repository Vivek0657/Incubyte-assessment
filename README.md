## Installation

Use npm to install the dependencies

```bash
    npm install
```

## Basic Usage

#### Open cypress

```bash
    npx cypress open
```

#### Run all features

```bash
    npx cypress run 
```

#### Run all features on specific browser

```bash
    npx cypress run --browser firefox
```

#### Run all features on headless mode

```bash
    npx cypress run --headless
```
